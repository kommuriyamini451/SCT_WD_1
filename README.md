# 🌐 SkillCraft Technology Landing Page - SCT_WD_1

Responsive landing page built as Task 1 for **SkillCraft Technology** Web Development Internship.

### 🔗 Live Demo
https://kommuriyamini451.github.io/SCT_WD_1/

## ✨ Features
- **Fully Responsive Design** - Mobile, tablet & desktop friendly
- **Smooth Scroll Navigation** - Fixed navbar with active link highlighting
- **Modern UI/UX** - Dark gradient theme with glassmorphism effects
- **Interactive Sections**
    - Home - Hero section with CTA
    - About - Company overview
    - Services - What SkillCraft offers
    - Contact - Form with validation
- **No Frameworks** - Built with pure HTML, CSS & JavaScript
- **Cross-Browser Compatible**

## 🛠️ Tech Stack
- **HTML5** - Semantic markup
- **CSS3** - Flexbox, Grid, Media Queries, Animations
- **JavaScript** - Smooth scroll, form validation, DOM manipulation
